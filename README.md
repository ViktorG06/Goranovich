# Goranovich

A multifunctional Discord bot focused on moderation, server utilities, and community management tools.

## About

Goranovich is a personal, one-person, long-term project built with Python and Discord.py.  
The goal of the bot is to provide useful moderation and server-management functionality while serving as a platform for experimenting with new features and improving backend/programming skills.

## Current Features

- Restricted word filtering
- Moderation utility commands
- Server-specific configuration support
- Database-backed settings
- Modular command/cog structure

## Planned Features

- Logging and moderation history
- Improved permissions system
- Music/community features
- Better configuration tools
- Web dashboard integration
- Performance and scalability improvements

## Technologies Used

- Python
- Discord.py
- SQLite / database integration
- Git & GitHub

## Project Status

Currently in active development.

This repository represents an ongoing learning and development project, so features, structure, and implementation details may change over time.

## Goals

This project is being developed to:
- Improve backend and software engineering skills
- Gain experience working with APIs and databases
- Practice scalable project structure
- Experiment with moderation and automation systems

## Setup

```bash
git clone <repo-url>
cd goranovich
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file and add your Discord bot token.

Run the bot:

```bash
python main.py
```

## Notes

Sensitive tokens, credentials, and private configuration files are not included in this repository.
